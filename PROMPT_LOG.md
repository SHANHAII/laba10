### Задание №7: Graceful Shutdown (Механизм плавного завершения)

#### Часть 1: Реализация на Go
**Промпт 1**
**Инструмент:** Gemini 3 Flash
**Промпт:** "Напиши HTTP-сервер на Go (используя стандартный net/http), который реализует Graceful Shutdown. Сервер должен перехватывать сигналы SIGINT и SIGTERM, выводить сообщение 'Завершение работы...' и корректно закрывать соединения с таймаутом в 5 секунд."
**Результат:** Код успешно перехватывает сигналы, но в первой версии отсутствовал контекст с дедлайном для функции Shutdown.

**Промпт 2**
**Промпт:** "Добавь в код использование context.WithTimeout для функции srv.Shutdown(ctx). Убедись, что после закрытия сервера основная горутина дожидается завершения процесса очистки перед выходом из main."
**Результат:** Реализовано корректное ожидание завершения всех процессов. Тест через Ctrl+C в терминале подтвердил работоспособность.
